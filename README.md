# Basic Html
A simple static website built with HTML, CSS, and JavaScript.

## Live Demo
https://hrs_124.oneapp.dev/

## Files
- index.html -> Main homepage
- tech.html -> Technology/about page
- styles.css -> Styling
- script.js -> JavaScript behavior
- images -> Visual assets

## Purpose
This project was created to understand the basics of frontend web development.

## How to run
1. Clone or download the repo
2. Open 'index.html' in your browser
